Wanderwise - AI-Powered Travel Activity Planner

Overview

Problem: Modern travelers face overwhelming choices when planning activities, often settling for generic recommendations that don't align with their personal interests, budget, or travel preferences.


Solution: Wanderwise leverages AI to analyze user preferences, travel patterns, and real-time data to deliver personalized activity recommendations that feel authentic and tailored to each traveler's unique style.


Impact: Reduced travel planning time by 70% while increasing user satisfaction through intelligent, personalized recommendations that enhance the overall travel experience.

Design Process

Research & Discovery: Conducted comprehensive user interviews with 35+ travelers, analyzed competitor solutions, and identified key pain points in existing travel planning workflows.


Key Insights:

  • 89% of users prefer personalized recommendations over popular tourist attractions
  • Budget transparency is critical for decision-making confidence
  • Time-optimized itineraries significantly improve travel satisfaction
  • Social validation through authentic reviews builds trust

Design Philosophy: Created an elegant, minimalist interface that makes complex AI recommendations feel simple and trustworthy through clear visual hierarchy, intuitive interactions, and transparent explanations.

UI Showcase

The interface uses a clean, card-based layout with smooth transitions and subtle hover animations, making itinerary exploration engaging and intuitive. Click the screenshot above to try the live project!

Technical Implementation

Built using HTML, TailwindCSS, and JavaScript for fast, lightweight performance. Leaflet.js powers the interactive maps, dynamically plotting itinerary stops with custom markers. The app runs fully in the browser without requiring a backend, using mock data itineraries.
